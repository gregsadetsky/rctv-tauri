# RCTV Tauri App

## how to dev/test

```
# In development:
npm run tauri build
./src-tauri/target/release/rctv-tauri --token YOUR_TOKEN_HERE

# Or after building for production:
./rctv-tauri --token YOUR_TOKEN_HERE
```
